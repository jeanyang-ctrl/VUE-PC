## 简介

**vue-shop**，✍电商后台管理系统用于管理用户账户，商品分类，商品信息、订单、数据统计等业务功能



## 快速开始

****

```bash
git clone https://github.com/jeanyang-ctrl/VUE-.git
```

代码下载后，用vscode打开后运行npm i 下载node-modules，这里用的是本地的数据库，所以一些数据展示可在readme文档可以初览。
